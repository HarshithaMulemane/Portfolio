# Harshitha Mulemane Lingaraja - Portfolio

A modern, responsive portfolio website built with HTML and TailwindCSS showcasing AI engineering expertise, NLP capabilities, and cloud data platform development.

## Features

- **Modern Design**: Minimalist 2026 design with clean typography and lots of whitespace
- **Mobile-First**: Fully responsive layout optimized for all devices
- **Dark/Light Mode**: Toggle between themes with persistent preference
- **Smooth Animations**: Subtle hover effects and section reveal animations
- **Project Filtering**: Filter projects by category (AI, Data, Cloud)
- **Fast Loading**: Optimized with TailwindCSS CDN and minimal dependencies
- **SEO Optimized**: Proper meta tags and semantic HTML

## Tech Stack

- **HTML5**: Semantic markup
- **TailwindCSS**: Utility-first CSS framework
- **JavaScript**: Vanilla JS for interactivity
- **Inter Font**: Modern typography from Google Fonts

## Sections

1. **Hero**: Compelling headline and key highlights
2. **About**: Professional story and expertise areas
3. **Projects**: Featured projects with filtering capability
4. **Skills**: Technical skills organized by category
5. **Experience**: Professional timeline
6. **Highlights**: Key achievements and impact
7. **Contact**: Call-to-action and contact links

## Color Palette

- **Primary**: Deep blue (#1e3a8a)
- **Secondary**: White (#ffffff)
- **Accent**: Light purple (#a855f7)
- **Accent Alt**: Teal (#14b8a6)
